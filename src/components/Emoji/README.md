Default Emoji

```
<div>
  <Emoji char="⏯" />
  <Emoji char="🐜" />
  <Emoji char="😝" />
</div>
```

For changing Emoji size just add `size` prop to component. Maximum size for emoji is 120;

```
<div>
  <Emoji char="⏯" size="30" />
  <Emoji char="🐜" size="60" />
  <Emoji char="😝" size="120" />
</div>
```
